Vi Commands for Mac

Command Modes

  • i - Insert mode
  • Esc - Command mode
  • :wq - Save and quit
  • :q! - Quit without saving

Movement

  • h - Move left
  • j - Move down
  • k - Move up
  • l - Move right
  • gg - Move to the beginning
  • G - Move to the end

Edit

  • x - Delete character
  • dd - Delete line
  • yy - Copy line
  • p - Paste
  • :m,n d - Delete multiple lines (replace m and n with line numbers)

Search and Replace

  • / - Search forward
  • ? - Search backward
  • n - Next match
  • N - Previous match
  • :s/search/replace/ - Replace

Save and Quit

  • :w - Save
  • :q - Quit
